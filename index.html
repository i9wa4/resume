1. 職務要約

北海道大学卒業後、オークマ株式会社で約2年Windowsアプリエンジニアを経験し、新光商事LSIデザインセンター株式会社で組込エンジニアとして約3年勤務しました。

その後株式会社シイエヌエス北海道にてデータエンジニアとして2年間データマート整備・機械学習ワークフロー構築・機械学習モデル運用業務に従事し、サブリーダーやリーダーを任されました。

2024年4月から現職である株式会社hacomonoにて一人データエンジニアとして従事しております。

2. 活かせる経験・スキル

職歴を通して要件定義・開発・テスト・運用・保守のいずれの工程も経験を積んでおります。

データエンジニアとしては Snowflake でのデータマート整備、Airflow や dbt を用いたデータパイプライン構築業務を行ったのち、自身で運用・保守まで行っております。機械学習ワークフロー構築・運用も経験済みです。

役割に関して、前職シイエヌエス北海道ではプロジェクトのリーダーやサブリーダーを任されており、現職 hacomono では一人データエンジニアとしてデータ基盤に対する責任を負っています。

7. 通常業務以外の活動

7.1. 自宅用ゲームサーバー構築

  • Google Compute Engine (Ubuntu) にてゲームサーバーを構築しました。Cloud Functions と GAS により Slack 経由でサーバーを起動・停止できる仕組みを導入しました。
  • 上記サーバーを構築後、コスト節約のために自宅マシンに Ubuntu をインストールし、Docker コンテナでゲームサーバーを構築しました。systemd を活用して AWS S3 への自動バックアップ機能やサーバーの自動アップデート機能を実装しました。
    • Repository: https://github.com/i9wa4/minecraft-bedrock-server-setup

7.2. Vim 活動

  • Vim の環境構築を通して各言語の環境構築の知見を得て、社内還元した経験があります。
  • Vim コミュニティのオンライン・オフラインのイベントに複数参加し Vim に関する技術情報収集や交流を行っています。
    • 参加イベント: https://connpass.com/user/19wa4/
  • Markdown 向けの簡易的な Formatter を Vim プラグインとして作成しました。TypeScript で作成しており Deno 環境にて動作します。
    • Repository: https://github.com/i9wa4/markdown-number-header.vim
